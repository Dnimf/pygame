#Descrição breve sobre o que é o projeto: 
Criado como projeto  da matéria de Desoft, Zelda da Shopee é a demonstração dos conhecimentos da linguagem de programação Python e da biblioteca pygame. Escolha do tema se deu pela inspiração do clássico da Nintendo, em especifico da Série de games The Legend of Zelda Minish Cap. 

# Como Iniciar
- Instalação do Python, disponível em:

https://www.python.org/downloads/release/python-3123/

- Compilador VSCode, disponível em: 

https://code.visualstudio.com/

- Após instalação dos executáveis, basta abrir os arquivo “jogo.py” e "arquivo antonio.py", e as pastas "Sprites" e "Sound effects" no vscode;

- Após abrir, instale a biblioteca pygame inserindo o seguinte comando no terminal do seu VSCode: %pip install pygame

- Por fim, rode o codigo do arquivo "Arquivo antonio.py"
